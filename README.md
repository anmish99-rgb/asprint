## Customization Options

### Add Client Logos
Replace `<div class="client-placeholder">AMUL MILK</div>` with:
```html
<img src="path/to/logo.png" alt="Client Name">
```

### Add Machine Images
Replace `<div class="service-image">🖨️ UV Printer</div>` with:
```html
<img src="path/to/machine.jpg" alt="Machine Name">
```

### Modify Colors
Main colors used:
- Primary Blue: #1e3c72, #2a5298
- Accent Orange: #ff6b35, #f7931e
- Purple: #667eea, #764ba2
- Background: #f8f9fa

Edit the contact details in the "Contact Section" (around line 580)
